# 📥 Social Inbox Triage | Replit Application
👉 Live App: https://sefket24-social-inbox-triage.streamlit.app/

This project was built for the Social Media Support role at Replit. It focuses on managing high-volume inbound messages across social channels—prioritizing conversations, categorizing issues, and responding efficiently while reducing manual triage work.

The goal is simple: show how social support workflows can scale without losing speed or clarity.
